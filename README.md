# Elite Surface Cleaning Website

A responsive static website for Elite Surface Cleaning, ready to host with GitHub Pages.

## Publish With GitHub Pages

1. Create a new GitHub repository.
2. Upload all files and folders from this project to the repository root.
3. Use `main` as the repository's default branch.
4. Open the repository's **Settings**, then select **Pages**.
5. Under **Build and deployment**, set **Source** to **GitHub Actions**.

The included workflow will publish the website after each push to `main`.

## Contact Form

The enquiry form uses FormSubmit and sends messages to:

`elitesurfacecleaninguk@gmail.com`

FormSubmit may send an activation email after the first form submission. Follow the link in that email to activate enquiries.

## Project Files

- `index.html` - website content
- `styles.css` - responsive design and branding
- `script.js` - mobile menu and comparison sliders
- `assets/` - logo, flyer and before/after images
